## Project Setup

- Clone this repository
- Install all dependencies with  
   `$ npm install`
- Run app in dev mode with  
   `$ npm run dev`
- Server: [http://localhost:3000/](http://localhost:3000/)
- Run tests via  
  `$ npm run test`
- To create a build ready for deploying run  
  `$ npm run build`
